web切图PS软件设置

2017.07.20 brtmay 切图  热度 ℃

photoshop设置web切图工作区

  1. 新建，设置宽度1920，高度1080，分辨路72，都调成像素，颜色模式RGB，背景设成透明，单击确定完成设置，存储为预设
  2. 移动工具设置：选项中的自动选择设置成图层，取消自动选择复选框。
  3. 视图设置，勾选显示->智能参考线，勾选标尺。
  4. 窗口设置，信息，字符，图层，历史记录。其他窗口全部关闭
  5. 单击信息面板选项：改像素，RGB，勾选文档尺寸。
  6. 编辑->首选项 单位标尺，单位都改成像素
  7. 窗口 工作区 设置为web切图，ok
