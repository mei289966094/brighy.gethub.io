Hexo的使用

2017.07.13 brtmay 技术  热度 ℃

本地环境搭建

git bash环境下输入
npm install -g hexo
hexo init
npm install
hexo g
hexo server
此时可以用locallhost:4000访问博客

config.yml中deploy的设置

deploy: 
  type: git
  repository: https://github.com/megmeet/megmeet.github.io.git
  branch: master

ssh keys的设置

检查ssh key是否存在                 
ls -al ~/.ssh
替换邮箱为注册github时的邮箱     
ssh-keygen -t rsa -C "angelen10@163.com"
ssh-agent -s
ssh-add ~/.ssh/id_rsa
 eval ssh-agent -s
ssh-add
复制sshkey
clip < ~/.ssh/id_rsa.pub
进github添加ssh key，最后测试是否连接成功 
ssh -T git@github.com

执行一次下面的语句

内容替换成自己的
 git config --global user.name "Firstname Lastname"
git config --global user.email "your_email@youremail.com"

部署静态网页

每次都要输入命令
hexo clean   
hexo generate  （或者hexo g）   
hexo deploy （或者hexo d）

绑定域名

添加一个无后缀的CNAME文件，
里面写上域名 www.sunoo.top,
再在域名提供商处加两条记录，
一条是带www的，一条不带www的，
ip地址ping自己的博客地址
